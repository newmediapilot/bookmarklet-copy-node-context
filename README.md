# bookmarklet-copy-node-context

Copies metadata of a selected page element—including selector.
> Paste it into your favorite interpreter

## Scripts

| Command           | Description       |
|-------------------|-------------------|
| `npm run dev`     | (build + serve)   |
| `npm run build`   | Build vite bundle |
| `npm run compile` | Compile out code  |

> Copy `out.js` into a bookmark field
